# Update Display Name

Verify that a user can update their display name through the profile page.

## Steps

1. Log in
2. Navigate to the profile page
3. Find the display name input field
4. Clear the field and type a new display name
5. Click the "Save" or "Update" button
6. Wait for the success toast or confirmation message
7. Refresh the page
8. Verify the display name field still shows the updated value

## Expected Results

- The display name input field is editable
- The field accepts a new display name value
- A success toast or confirmation message appears after saving
- After page refresh, the display name field retains the updated value
- The updated name is persisted to the database
