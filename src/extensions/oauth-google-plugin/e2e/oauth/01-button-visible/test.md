# Google OAuth Button Visible on Login Page

Verify that the oauth-google-plugin renders a "Sign in with Google" button on the login page.

## Steps

1. Navigate to the login page
2. Wait for the page to fully load
3. Verify a "Sign in with Google" button is visible in the OAuth section
4. Verify the button is clickable and not disabled

## Expected Results

- The login page loads with the standard email/password form
- A "Sign in with Google" button is visible in the OAuth section above the login form
- The button has the Google branding or icon
- The button is in an enabled, clickable state
